## üöÄ Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository or use this directory:
   ```bash
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit the local URL shown in the terminal (typically `http://localhost:5173`)

### To Further Customize

1. **Add Your Photo** - Replace placeholder in `src/components/About.jsx`
2. **Update Projects** - Modify `src/components/Projects.jsx` to add more projects
3. **Adjust Skills** - Fine-tune skill levels in `src/components/Skills.jsx`
4. **Change Experience** - Update timeline in `src/components/Experience.jsx`

### Color Scheme

The color scheme is defined in `src/index.css` using CSS variables. Modify these to change the overall look:

```css
:root {
  --primary: #6366f1;
  --secondary: #ec4899;
  --accent: #14b8a6;
  /* ... other colors */
}
```

### Adding Your Photo

Replace the placeholder in `About.jsx` with an actual image:

```jsx
<div className="image-placeholder">
  <img src="/your-photo.jpg" alt="Your Name" />
</div>
```
